<h1>我的毕业设计--电影票购票系统  </h1>
界面设计很Low，对颜色布局设计很菜，没感觉，功能简单，对于毕设足以。    
其中包括一些部分功能的小Demo测试，比如官方侧滑菜单、地址分类导航、自定义手势密码、百度定位、图灵机器人聊天问答等。  
模块功能持续更新中...
